### 这里是一些小爬虫集合

---

每个平台对应一个文件，每个文件里面有一个`get(url: str)`函数 统一形式如下(里面使用 f-string 需要 python3.6+)：

```python
"""
Args:
    url: str
Returns:
    data: dict
      |_ {
            title: str,
            author: str,
            audioName: str,
            videoName: str,
            imgs: List[str],
            audios: List[str],
            videos: List[str],
            text: str,
            msg: str
         }
Tips:
    data里面的各个字段只有当爬取到相关内容时才会存在，除了msg(不过这个没啥大用)
    ☆ 爬取未成功也会返回data，而且不一定为空
"""
```
